# 📚 Sistema de Biblioteca (Python + MySQL)

Sistema de gerenciamento de biblioteca desenvolvido em Python com MySQL, rodando via terminal. Permite o controle de livros com operações básicas de CRUD.

---

## 🎯 Objetivo

Projeto desenvolvido para fins de estudo, com foco em praticar lógica de programação, CRUD (Create, Read, Update, Delete), integração entre Python e banco de dados MySQL e organização de código em funções.

---

## ⚙️ Funcionalidades

- Cadastro de livros  
- Listagem de livros  
- Busca de livros  
- Atualização de registros  
- Exclusão de livros  

---

## 🛠️ Tecnologias utilizadas

- Python 
- MySQL  
- mysql-connector-python  

---

## ▶️ Como executar

Instale a dependência:

pip install mysql-connector-python

Configure as credenciais de acesso ao banco no código e execute:

python main.py

Projeto desenvolvido exclusivamente para fins de estudo.
